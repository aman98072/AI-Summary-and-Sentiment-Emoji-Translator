# AI Summary & Sentiment Emoji Translator

A full-stack AI application that analyzes long text and returns:

- A concise **4-line summary**
- Overall **sentiment analysis**
- **Emoji-based highlighted sentences**

The backend uses **FastAPI** and **OpenAI API**, while the frontend is built with **React + TypeScript (Vite)**.

---

# Tech Stack

## Backend
- Python
- FastAPI
- OpenAI API

## Frontend
- React 18
- TypeScript
- Vite
